# React Native Cadastro de Bueiros

Projeto para cadastro de bueiros para disciplina de coding

## 🚀 Começando

Copiar esse repositório na sua máquina

### 📋 Pré-requisitos

node.js 12.x

```
Dar exemplos
```

### 🔧 Instalação



Rodar os comandos em ordem:

```
npm install
```

E depois:

```
expo start
```

Se atente a deixar o backend e frontend rodando no mesmo ip, e quando for testar o expo no celular, deixar o celular na mesma rede que o computador está.
